## How to use

### Browser

```js
<script src="lib/EventListener.js">
<script>
console.log( new EventListener() );
</script>
```

### WebWorkers

```js
importScripts("lib/EventListener.js");

console.log( new EventListener() );
```

### Node.js

```js
var EventListener = require("lib/EventListener.js");

console.log( new EventListener() );
```
